# In-app walkthrough

On **Insights** (or Home), click **Start walkthrough**.

A dock stays at the bottom with **Previous**, **Next**, and **Cancel**.
The current UI target stays visible; the rest of the screen is dimmed.

## Tour path

1. Upload dropzone  
2. Knowledge equipment list  
3. Equipment history  
4. Ask AI question box  
5. Answer / action area  
6. Insights coverage  
7. Open issues  

Ask AI is **universal** — mention equipment in the question (e.g. P-102). You do not need to select an asset first on Knowledge.

## Manual demo (no tour)

1. Upload a short text file mentioning `P-102`  
2. Knowledge → Search “seal” or “vibration” → open a result  
3. Ask AI → “Why did Pump P-102 fail?”  
4. Insights → review conflicts and gaps
